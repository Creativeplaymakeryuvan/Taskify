# 🚀 React + TypeScript + Vite Project

A modern and fast web application built using **React**, **TypeScript**, and **Vite**. This project is designed with performance, scalability, and clean architecture in mind.

---

## ✨ Features

* ⚡ Lightning-fast development with Vite
* 🔒 Type safety using TypeScript
* 🧩 Reusable and modular components
* 🎯 Clean and maintainable code structure
* 📱 Responsive design
* 🔄 Easy state management
* ⚡ Drag and Drop feature

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Language:** TypeScript
* **Build Tool:** Vite
* **Styling:** CSS / react-icons / @hello-pangea/dnd 

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Creativeplaymakeryuvan/Taskify.git
cd your-repo
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Running the Project

Start the development server:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
